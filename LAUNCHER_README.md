# ⚡ ALL Python Apps — Launcher

A premium app launcher that displays all 20 Python applications in a beautiful grid with big icons, names, and descriptions. Click any card to instantly launch the app.

---

## ✨ Features

- **Visual app grid** — All 20 apps displayed with colored icons and descriptions
- **One-click launch** — Click any card to start the application
- **Search & filter** — Find apps instantly by name or description
- **Running status** — See which apps are currently active with green indicators
- **Process tracking** — Monitors launched app PIDs and auto-detects when they close
- **Streamlit support** — Automatically uses `streamlit run` for dashboard apps
- **Premium dark UI** — Deep obsidian theme with accent colors and hover effects
- **Responsive grid** — Adapts columns based on window width
- **Scrollable** — Mouse wheel support for the full app list

---

## 🚀 How to Run

```bash
python launcher.py
```

> No additional dependencies required — uses only Python standard library (tkinter).
