# 📈 Attendance Dashboard — Streamlit Data Visualizer

A Streamlit-based interactive dashboard for visualizing attendance data from Excel files, featuring a hacker-themed UI with charts and filters.

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)
![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Mac%20%7C%20Linux-0078D6)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Features

- **Interactive charts** — Bar, line, and pie charts for attendance data
- **Excel file import** — Load `.xlsx` attendance records
- **Filter by subject** — Drill down into specific courses
- **Attendance percentage** — Auto-calculated attendance rates
- **Hacker-themed UI** — Custom dark styling with green accents
- **Real-time updates** — Streamlit reactive interface

---

## 🛠️ Setup & Installation

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Dashboard
```bash
streamlit run dashboard.py
```

### Step 3: Upload Data
Upload an attendance Excel file (`.xlsx`) through the Streamlit interface.
